# Результаты поиска: Монолитный PHP проект

## 1. BookStack

- **Ссылка на GitHub:** [https://github.com/BookStackApp/BookStack](https://github.com/BookStackApp/BookStack)
- **Описание:** Платформа для создания и управления документацией и вики-базами знаний.
- **Соответствие требованиям:**
    - **Тип:** Классическое монолитное приложение.
    - **Технологии:** Построено на PHP с использованием фреймворка Laravel.
    - **Источник:** Публичный репозиторий на GitHub.
    - **Комментарий:** Отличный кандидат для первой итерации развертывания. Проект имеет понятную структуру и хорошую документацию.

## 2. Akaunting

- **Ссылка на GitHub:** [https://github.com/akaunting/akaunting](https://github.com/akaunting/akaunting)
- **Описание:** Бесплатное open-source программное обеспечение для бухгалтерского учета, предназначенное для малого бизнеса.
- **Соответствие требованиям:**
    - **Тип:** Монолитное приложение.
    - **Технологии:** PHP, фреймворк Laravel.
    - **Источник:** Публичный репозиторий на GitHub.
    - **Комментарий:** Представляет собой реальное бизнес-приложение, что делает его полезным для отработки практических навыков развертывания.

# Результаты поиска: Монолитный Python проект

## 1. Saleor

- **Ссылка на GitHub:** [https://github.com/saleor/saleor](https://github.com/saleor/saleor)
- **Описание:** Высокопроизводительная, компонуемая, headless commerce API платформа электронной коммерции.
- **Соответствие требованиям:**
    - **Тип:** Монолитное приложение (бэкенд на Django).
    - **Технологии:** Python (Django), GraphQL.
    - **Источник:** Публичный репозиторий на GitHub.
    - **Комментарий:** Современная платформа электронной коммерции, позволяющая изучить развертывание сложных приложений на Python.

## 2. Wagtail

- **Ссылка на GitHub:** [https://github.com/wagtail/wagtail](https://github.com/wagtail/wagtail)
- **Описание:** Ведущая система управления контентом (CMS) с открытым исходным кодом, построенная на Django.
- **Соответствие требованиям:**
    - **Тип:** Монолитное приложение.
    - **Технологии:** Python (Django).
    - **Источник:** Основной репозиторий разработки находится на GitHub.
    - **Комментарий:** Отличный, широко используемый и активно развиваемый проект. Идеально подходит для изучения развертывания сложных Django-приложений, так как является реальным продуктом, а не демонстрацией.

# Результаты поиска: Микросервисный проект 
## 1. Online Boutique (microservices-demo)

- **Ссылка на GitHub:** [https://github.com/GoogleCloudPlatform/microservices-demo](https://github.com/GoogleCloudPlatform/microservices-demo)
- **Описание:** Облачное демонстрационное веб-приложение для электронной коммерции, состоящее из 11 микросервисов.
- **Соответствие требованиям:**
    - **Тип:** Микросервисное приложение.
    - **Технологии:** Kubernetes, gRPC, Istio, Skaffold. Сервисы написаны на разных языках (Go, Java, Python, C#, Node.js), что делает его отличным полиглотным примером.
    - **Источник:** Публичный репозиторий на GitHub.
    - **Комментарий:** Эталонный проект от Google, специально созданный для демонстрации развертывания, мониторинга и управления микросервисами в облачной среде. Идеально подходит для всех задач Фазы 2.
